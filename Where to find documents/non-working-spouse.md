# Non-Working Spouse / Low-Income Spouse Confirmation

**What it is**: Used to claim a credit point for a non-working spouse (Section 37 / סעיף 37) when the breadwinner supports them.

## Where to find it

- **Spouse's Form 106 (טופס 106)** — if there was any partial income.
- **Declaration / affidavit** stating the spouse did not work in the tax year.
